Code format is only one part

A referral code can fail because of format, account eligibility, region, expired campaign timing, or bonus conditions. Do not assume every failure is a typo.

Terms before reward

Users should read turnover and eligibility wording before expecting a bonus. Hidden or unclear terms are a risk signal.

Evidence for support

Save code source, time, account status, and the exact message shown after applying the code.

This page uses K1-only public evidence collected on 2026-06-13. B9 is not used as K1 evidence.

Action checklist

  1. Check code source and format.
  2. Read eligibility and turnover conditions.
  3. Save the error message.
  4. Avoid public posting of account identifiers.

FAQ

Why would a referral code not work?

Possible causes include wrong format, expired campaign, account eligibility, or unmet bonus terms.

Does a referral code guarantee a bonus?

No. Codes are subject to conditions, timing, and account eligibility.

What should I save before asking support?

Code source, time, account status, and the exact message shown.
